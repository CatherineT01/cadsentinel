JIT Cylinders H Series Drawing Standard
Version 1.0

1. TITLE BLOCK REQUIREMENTS

1.1 Model Code
The Materials field must contain a valid JIT H Series model code in the format:
[Series]-[Mount]-[Bore]X[Stroke]X[Rod]-[RodThread]-[Cushion]-[Port]-[Seal]-[Special]
Example: H-MT4-4X65X2-1-NC-S-V-S

1.2 Drawing Number
The drawing number must follow the format: [8 digits] ASSEMBLY
Example: 20002411 ASSEMBLY

1.3 Customer Field
The Customer field must be populated and must not be empty.

1.4 Sales Order Field
The Sales Order field must be populated and must not be empty.

1.5 Revision Table
The revision table must contain at least one entry. The first entry must be ORIGIN.

2. STANDARD NOTES REQUIREMENTS

2.1 Dimension Units
Drawings must include the note: -Dimensions are in Inches

2.2 Metric Brackets
Drawings must include the note: -[ ] Indicates Millimeters

2.3 Chamfer Size
Drawings must include the note: -Chamfer Size = .03"x45 deg.

2.4 Diameter Concentricity
Drawings must include the note: -All Diameters within .002" TIR

2.5 Hole Tolerance
Drawings must include the note: -Holes less than 0.500" +0.003 -0.000

2.6 Three Place Decimal Tolerance
Drawings must include the note: -Decimal: x.xxx 0.005

2.7 Two Place Decimal Tolerance
Drawings must include the note: -Decimal: x.xx 0.015

2.8 Angular Tolerance
Drawings must include the note: -Angular: x 0.5

3. CYLINDER SPECIFICATION REQUIREMENTS

3.1 Bore Note
The NOTES block must contain a BORE entry specifying the cylinder bore diameter.
Example: BORE - 3.250

3.2 Stroke Note
The NOTES block must contain a STROKE entry specifying the cylinder stroke length.
Example: STROKE - 10

3.3 Rod Note
The NOTES block must contain a ROD entry specifying the piston rod diameter.
Example: ROD - 1.000

3.4 Ports Section
The NOTES block must contain a PORTS section with at least one port entry.
Each port entry must specify port location, size, and position.
Example: CEH - .500 NPT - PP1

3.5 Valid Bore Size
The bore diameter must be one of the valid JIT H Series bore sizes:
1.5, 2.0, 2.5, 3.25, 4.0, 5.0, 6.0, 7.0, 8.0 inches.

3.6 Valid Mounting Style
The mounting style code in the model code must be one of the valid JIT mounting styles:
MHF, MCF, ME5, ME6, MF1, MF2, MF5, MF6, MP1, MP2, MP3, MPU3,
MS1, MS2, MS3, MS4, MS7, MT1, MT2, MT4, MX0, MX1, MX2, MX3

4. DIMENSIONAL REQUIREMENTS

4.1 Dimension Units
All dimensions must be specified in inches.
Metric equivalents in brackets [ ] are acceptable but not required.

5. PROPRIETARY STATEMENT

5.1 Confidentiality Notice
Drawings must include the JIT Industries proprietary and confidential statement.
